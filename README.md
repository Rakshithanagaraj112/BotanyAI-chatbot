# 🌿 Botany AI Chatbot

Botany AI Chatbot is a simple AI-based chatbot that answers questions related to plants and botany. It provides information about plant types, plant care, and basic botany concepts through an interactive chat interface.

The project is built using **Python for the backend** and **HTML for the chatbot interface**.

## 🛠 Technologies Used

* Python
* HTML
* css
* javaScript

##  Project Files

* `app.py` – Python backend that runs the chatbot
* `index.html` – Web interface for chatting with the bot

##  How to Run the Project

1. Download or clone the repository.
2. Open **Command Prompt**.
3. Navigate to the project folder.

Example:
cd botany-ai-chatbot

4. Run the Python file:

python app.py

5. Open the HTML file in your browser to interact with the chatbot.

## Purpose

This project demonstrates how a simple AI chatbot can be created to help users learn about botany and plants.
